◆ pid_state_e

enum pid_state_e

PID state.

A PID can be either free (no process is using it), ALIVE (some running process is using it), or ZOMBIE (a zombie process is using it).

Enumerator
FREE 

The PID is free and available.

ALIVE 

The PID is given to a process.

ZOMBIE 

The PID is held by a zombie.

PCB* get_pcb ( Pid_t  pid)

Get the PCB for a PID.

This function will return a pointer to the PCB of the process with a given PID. If the PID does not correspond to a process, the function returns NULL.
